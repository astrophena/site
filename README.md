# [astrophena.name](https://astrophena.name)

This is my personal website.

## Serving locally

You need the latest [Go] installed.

```sh
$ git clone https://github.com/astrophena/site
$ cd site
$ go tool serve
```

Open http://localhost:3000 in your browser.
